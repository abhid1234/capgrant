A fleet of agents runs with too much ambient authority. capgrant is the access checkpoint: an issuer hands an agent a scoped, expiring capability grant — [ fs.write : src/** ] for the next 30 minutes — and before the agent acts, it checks the gate. Below is a working console: issue grants, run the real check() through the turnstile, delegate a strict subset, try to escalate and get blocked, revoke a parent and watch the cascade, then audit a batch. Every decision is the real, unmodified library running client-side.
